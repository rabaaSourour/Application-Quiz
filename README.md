📚 Application Quiz

📖 Description

Application web interactive permettant aux utilisateurs de tester leurs connaissances à travers des quiz.
L'application propose un système de connexion, d'inscription et de sélection de questions selon leur niveau de difficulté.


🚀 Fonctionnalités
🔐 Authentification (connexion / inscription)
📝 Gestion des utilisateurs
❓ Quiz dynamique basé sur un fichier JSON
🎯 Sélection du niveau de difficulté des questions
📊 Affichage des résultats
🎨 Interface utilisateur moderne (HTML, CSS, JavaScript)

🗂️ Structure du projet
Application-Quiz/
│── index.html          # Page principale du quiz
│── connexion.html      # Page de connexion
│── enregistrer.html    # Page d'inscription
│── questions.json      # Base de données des questions
│── script.js           # Logique principale du quiz
│── login.js            # Gestion de la connexion
│── enregistrer.js      # Gestion de l'inscription
│── styles.css          # Style global
│── login.css           # Style page connexion
│── registre.css        # Style page inscription
│── .vscode/            # Configuration VS Code

🛠️ Technologies utilisées
HTML5
CSS3
JavaScript (Vanilla JS)
JSON
